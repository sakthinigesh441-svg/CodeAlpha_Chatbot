# CodeAlpha - Basic Chatbot

## Task 4 - Python Programming Internship

This is a simple rule-based chatbot developed in Python as part of the CodeAlpha Python Programming Internship.

## Features

- Responds to greetings
- Answers basic questions
- Handles thank you messages
- Provides help information
- Tells programming jokes
- Responds to weather and time queries
- Ends conversation using "bye"

## Technologies Used

- Python 3
